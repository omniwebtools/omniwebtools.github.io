What is URL Encoding?

URLs can only safely contain a limited set of characters. Spaces, symbols like &, ?, #, and non-English characters can break a URL or change its meaning. URL encoding converts these characters into a safe format, like turning a space into %20.

URL decoding does the opposite — it converts an encoded string back into normal, readable text.

When Would You Need This?

This is commonly used by developers when building links that include search queries, passing data between web pages, or working with APIs that require encoded parameters in their requests.

Frequently Asked Questions

What does %20 mean?

It's the encoded version of a space character. The percent sign followed by two characters represents the hexadecimal code of the original character.

Is my text stored anywhere?

No. Encoding and decoding both happen instantly in your browser. Nothing is sent to a server.

What happens if I try to decode normal text?

If there's nothing to decode, the output will simply look the same as the input.
